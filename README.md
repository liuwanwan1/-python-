# 图书信息管理系统

> Python 程序设计期末实验作业

## 一、项目简介

本项目是一个基于 Python 实现的**简易图书信息管理系统**，采用模块化设计，支持图书信息的录入、删除、查询和格式化输出等功能。

## 二、功能介绍

| 功能编号 | 功能描述 | 实现状态 |
|---------|---------|---------|
| 1 | 录入图书信息（书名、编号、作者、价格、分类） | ✅ 已完成 |
| 2 | 根据图书编号删除对应图书信息 | ✅ 已完成 |
| 3 | 根据图书编号查询单本图书详情 | ✅ 已完成 |
| 4 | 格式化输出全部图书信息，排版整洁 | ✅ 已完成 |

## 三、技术特点

- **数据存储**：使用 Python 列表存储所有图书数据，每本图书以字典形式保存
- **模块化设计**：各功能独立封装为函数，结构清晰
- **输入验证**：对价格、编号等进行合法性校验
- **无全局变量**：所有数据通过参数传递，符合编程规范

## 五、运行方法

### 环境要求
- Python 3.x

### 运行命令
```bash
python book_manager.py
```

### 操作说明
1. 运行程序后，显示主菜单
2. 输入对应数字选择功能：
   - `1` — 录入图书信息
   - `2` — 删除图书信息
   - `3` — 查询图书信息
   - `4` — 显示全部图书信息
   - `0` — 退出系统
